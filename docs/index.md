# M2 NewSpace : Analyse de données

![Logo](img/M2NS_logo.png)

_Cours conçu pour les étudiants du M2 "NewSpace" de l'Université de Versailles Saint-Quentin (UVSQ), dans le cadre de l'UE "Traitement et analyse de données instrumentales"_

---

## Objectifs du cours

Ce cours portera sur **l'analyse de données avec Python** en utilisant des méthodes basiques de **classification automatique**.

L'objectif est de vous faire découvrir les méthodes de base pour la classification **supervisée** et **non-supervisée**, avec des applications sur des données instrumentales.
Nous suivrons les étapes suivantes :

- Importation des données à partir de fichiers TIFF et CSV.

- Une exploration du contenu des données.

- L'application de méthodes de classification supervisée ou non-supervisée.

- L'étude des performances des modèles obtenus.

- Une discussion sur la généralisation.

Ce cours sera divisé en 3 temps :

- Un 1er **tutoriel** de classification supervisée, appliquée à une image du satellite Sentinel 2, que vous trouverez sur ce site web.

- Un 2nd **tutoriel** de classification non-supervisée (ou "partitionnement"), appliquée à un catalogue d'exoplanètes, que vous trouverez également sur ce site web.

- Un **projet évalué** lors duquel vous mettrez en application ce que vous avez appris, sur un nouvel exemple.
